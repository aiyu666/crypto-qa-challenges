# crypto-qa-challenges

# Installation
1. Install poetry
    ```
    brew install poetry
    ```
2.  Install the dependency package
    ```
    poetry install
    ```
3. Active the virtual env
    ```
    poetry shell
    ```
4. cd to project
    ```
    cd challenges_API
    ```
    or
    ```
    challenges_app_UI
    ```

## Check code with pre-commit
### Load pre-commit config
    ```
    pre-commit install
    ```
### Check all of files
    ```
    pre-commit run --all-files
    ```
